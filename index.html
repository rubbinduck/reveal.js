Quick Rundown of Definitions and Notation

We denote by $\mS^d$ the vector space of symmetric $d\times d$ matrices, by $\mS^d_{\geq 0}$ (resp. $\mS^d_{>0}$) the cone of positive semidefinite (resp. definite) matrices and by $\IR^d_{\geq 0}$ (resp. $\IZ^d_{\geq 0}$) the set of componentwise non-negative (integral) vectors.

Let \[ \cop^d = \bigg\{ Q\in\mS^d: x^\intercal Qx \geq 0 \text{ for all } x\in\IR^d_{\geq 0} \bigg\} \] denote the cone of copositive matrices of order $d$ .

\[ \minC Q = \min_{x\in\IZ^d_{\geq 0}}Q[x] \] \[ \MinC Q = \bigg\{ x\in\IZ^d_{\geq 0}: Q[x] = \minC Q \bigg\} \] For matrices $Q\in\oint\cop^d$ we have \[ \minC Q > 0 \text{ and } |\MinC Q| < \infty. \]

A matrix $Q\in\oint\cop^d$ is called perfect copositive , if the set \[ \mV(Q) = \bigg\{ xx^\intercal: x\in\MinC Q \bigg\} \] has a non-empty interior in $\mS^d$.

Connecting the Classics with the Newfangled
Theorem:
Every classically perfect matrix is artihmetically equivalent to a classically perfect matrix, which is also perfect copositive.
Classifying all positive semidefinite perfect copositive matrices
Theorem:
Let $P\in\obd\mS^n_{\geq 0}\cap\oint\cop^n$ with $\ork P=k \geq 2$. Then $P$ is perfect copositive if and only if there exists a $U\in\text{GL}_n(\IZ)$ and a perfect copositive $\widetilde{P}\in\mS^K_{>0}$, such that \[ U^\intercal PU = \begin{pmatrix} 0 & 0 \\ 0 & \widetilde{P} \end{pmatrix}. \]

Finiteness Results

The classic one:

Theorem: (A-M Bergé ?)
For any $Q_0\in\mS^n_{>0}$, constant $C>0$ and given minimum $m$, there are only finitely many classically perfect matrices $Q$ with minimum $m$, such that $\langle Q, Q_0 \rangle \leq C$.

Does not hold for any matrix not in $\mS^n_{>0}$?

The new (partial) one:

Theorem:
For any $P_0\in\oint\cp^n$, constant $C>0$ and given copositive minimum $m$, there are only finitely many perfect copositive matrices $P$ with copositive minimum $m$, such that $\langle P, P_0 \rangle \leq C$.

Does hold for matrices outside of $\oint\cp^n$!
